Ứng Dụng Trong Thực Tế

Parabol & Đồ Thị

Hình dạng cơ bản của đồ thị hàm số bậc hai, với đỉnh và trục đối xứng đặc trưng.

y = ax² + bx + c

Vật Lý - Ném Xiên

Quỹ đạo của vật thể khi ném xiên trong trọng trường, ứng dụng trong thể thao.

h = v₀t - ½gt²

Kinh Tế - Tối Ưu Lợi Nhuận

Tìm mức sản xuất tối ưu để đạt lợi nhuận cao nhất trong kinh doanh.

P = -ax² + bx - c

Kỹ Thuật - Chảo Parabol

Thiết kế chảo parabol, anten vệ tinh và các công trình kỹ thuật có hình parabol.

y = ax² + c

Tối Ưu Hóa Đời Sống

Tìm giá trị tối ưu trong các bài toán thực tế như tiết kiệm chi phí, thời gian.

f(x) = ax² + bx + c

Góc Toán - Phân Tích

Biệt thức Delta, đỉnh parabol, trục đối xứng và các tính chất toán học.

Δ = b² - 4ac

Bài Tập Nhanh

Bài 1: Quỹ Đạo Bóng Rổ

Một quả bóng được ném từ độ cao 2m, đạt độ cao tối đa 45m tại vị trí 45m từ điểm ném. Tìm phương trình quỹ đạo và khoảng cách bóng chạm đất.

Lời giải:

Bước 1: Đặt gốc tọa độ tại điểm ném

Phương trình: h(x) = ax² + bx + c

Điều kiện:

  • h(0) = 2 → c = 2
  • h(45) = 45 (đỉnh) → đỉnh tại x = 45
  • Đỉnh: x = -b/(2a) = 45

Kết quả: h(x) = -0.02x² + 1.8x + 2

Chạm đất khi: h(x) = 0 → x ≈ 91.1m

Bài 2: Tối Ưu Lợi Nhuận

Một công ty có hàm lợi nhuận P(x) = -2x² + 300x - 8000 (x là số sản phẩm). Tìm số sản phẩm cần sản xuất để đạt lợi nhuận tối đa.

Lời giải:

Phương trình: P(x) = -2x² + 300x - 8000

Tìm đỉnh parabol:

  • a = -2, b = 300, c = -8000
  • x = -b/(2a) = -300/(2×(-2)) = 75

Lợi nhuận tối đa:

P(75) = -2(75)² + 300(75) - 8000 = 3250

Kết luận: Sản xuất 75 sản phẩm để đạt lợi nhuận tối đa 3,250 đơn vị tiền tệ

Bài 3: Thiết Kế Cầu Vòm

Một cầu vòm có nhịp 60m, độ cao tối đa 20m tại giữa nhịp. Viết phương trình mô tả hình dạng vòm cầu và tính độ cao tại vị trí 15m từ chân cầu.

Lời giải:

Đặt hệ tọa độ: Gốc tại chân cầu trái

Điều kiện:

  • h(0) = 0, h(60) = 0 (chân cầu)
  • h(30) = 20 (đỉnh vòm)

Phương trình: h(x) = ax² + bx + c

Giải hệ: c = 0, a = -1/45, b = 4/3

Kết quả: h(x) = -x²/45 + 4x/3

Tại x = 15m: h(15) = -225/45 + 60/3 = 15m
